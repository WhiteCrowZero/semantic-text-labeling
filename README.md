# Semantic Text Labeling & Sentiment Platform
语义驱动文本情感与标签平台

## 项目简介

本项目是一个围绕「文本情感分析」与「智能打标签」构建的语义工作流平台，核心流程为：

句向量（Embeddings） → 情感分析 / 聚类 → 大语言模型打标签 → 人工审核

通过统一的句向量表示和大语言模型能力，平台提供两条主线能力：

1. **文本情感分析服务**  
   - 基于句向量与情感分类模型，对文本进行情绪极性判定（正向/负向/中性等）。  
   - 支持批量分析与在线 API 调用，适用于评论监控、用户反馈分析等场景。

2. **智能标签生成与审核服务**  
   - 利用聚类结果与大语言模型，为文本自动生成业务标签（主题标签、意图标签等）。  
   - 通过人工审核环节对标签进行校正和确认，形成可复用的高质量标注数据。

底层实现通过统一的工作流串联各模块：

- `Embeddings`：将文本映射到统一语义空间，为后续情感分析与聚类提供基础表示。  
- `Sentiment / Clustering`：分别完成情绪识别和文本结构聚类。  
- `LLM Tagging`：调用大语言模型在语义基础上自动生成标签。  
- `Human Review`：人工在前端界面审核与修改标签，形成闭环数据资产。

项目同时提供后端 API 与前端展示（课程作业需求），既可用于课堂演示，也可作为后续文本挖掘项目的基础工程骨架。
